An easy to use Wordpress favicon plugin.

== Description ==

A very simple favicon plugin. You can upload any size image and the plugin will resize it, convert it to an .ico file, and add the necessary html code to the header. 

== Installation ==

1. Upload `wp-favicon.php`, `wp_favicon_admin_page.php`, and `class-php-ico.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress.
